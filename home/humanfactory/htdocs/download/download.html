To setup the game, just follow these four easy steps:

Check your spec.

Currently this game is for PCs with Windows 95, 98, 98SE, ME, 2000, XP and NT 4.0 (with Service Pack 6) only.
We recommend using a PII 500 Mhz machine or above with a 3D Graphics Card.
We also advise using Internet Explorer 5 or above.

Install the Virtools 2.5 plugin from the Virtools website.

Download the .zip file and unzip it.
Click on "humanfactory.html" and wait for 30 secs until the content has loaded (this time will vary depending on the computer).

You are now ready to play the game!

If you have a broadband connection you might want to run the game directly. The game will open in a popup window. It may take a few minutes to load.

In Schools:
To use the game on a school network the plugin should be installed using an Administrator account. The game can then be played with restricted user account. The Virtools website gives further information about the plugin.

Trouble Shooting:
We recommend using a PII 500 Mhz machine or above with a 3D Graphics Card. If you do not have the minimum specification hardware, the game is likely to run very slowly.

If there are problems with the quality of the rendering, you can change the Render Engine by right clicking in the browser windows and opening the 'Settings' panel.
